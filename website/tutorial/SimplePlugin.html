从0开始制作一个简单的插件

插件在西奥小镇游戏中扮演了重要的角色，极大的丰富了游戏的内容，创造了无限可能。关于制作插件，其实并不难，简单的插件包含贴图和JSON代码，更高级的还包含Lua代码，着色器中使用GLSL着色器语言。本文主要介绍贴图制作和代码。

贴图

要制作贴图，首先你需要绘图软件，国内比较多人使用 Pixel Studio，你可以在应用商店搜索或加入一些群聊来下载它，一些绘画软件也是不错的选择。


紧接着呢你还需要下载一个建筑基座，用于绘图，你可以前往论坛下载，下载时最好打开图片再下载喔：点击此处跳转


关于绘制建筑，你可以观看上面的视频来获取一些灵感，或浏览这篇论坛文章来获取更多灵感：点击此处跳转


代码

关于代码，开发者JustAnyone提供了一个插件代码制作网站（点击此处跳转），你可以借助这个网站快速制作代码。


当然，你也可以看这里的教程。首先你还是需要一个JSON编辑器，不过对于安卓和桌面端平台我并不清楚什么软件比较好，但对于iOS平台，你可以使用 Koder 软件。


在一切准备好之后，我们开始编写吧！那么这里我先提供一个例子/模板：

例子

[
{
"id": "$Example",
"type": "type",
"title": "标题",
"text": "描述",
"frames": [{"bmp": "image.png"}],
"height": 1,
"width": 1
}
]

那么好，这就是一个最基础的代码模版，下面我会对其中的内容一一讲解。
